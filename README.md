# Microsoft Sentinel Security Lab

A hands-on lab environment focused on building and validating security detection engineering workflows, log ingestion, and incident response tracking using Microsoft Sentinel.

## Repository Structure
* `scripts/`: Contains attack simulation and automation scripts (such as the PowerShell brute-force loop).
* `detections/KQL/`: Stores custom KQL analytics rules and queries used in the workspace.
* `docs/`: Architecture diagrams, write-ups, and screenshots.
